10_Reference_Board

USAGE NOTES
- Use the primary logo when there is enough horizontal space.
- Use the secondary horizontal logo for website nav bars and email signatures.
- Use the stacked logo for square/centered layouts.
- Use the icon mark for favicons, watermarks, stamps, and social avatars.
- Use monochrome versions only when color reproduction is limited.
- Keep clear space around the logo; do not stretch, rotate, recolor, add shadows, or place it on a busy background.
